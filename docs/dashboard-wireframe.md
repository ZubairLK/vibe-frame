## Components:
- Top navigation bar with Preview, Share, and Sign Out buttons
- Left sidebar with navigation:
  - Home (active)
  - Links
  - Stats
  - Settings
- Main content area with:
  - Editable handle field
  - List of current links with:
    - Title and URL fields (editable)
    - Reorder buttons (up/down arrows)
    - Delete button (trash icon)
  - "Add Another Link" button
  - "Save Changes" button
  - Preview link button showing the live page URL
- Clean layout with clear visual hierarchy
